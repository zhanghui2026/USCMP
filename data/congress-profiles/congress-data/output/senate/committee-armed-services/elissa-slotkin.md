## 2025年国情咨文回应

2025年3月，Slotkin 被选为民主党全国性回应发言人，在特朗普的国情咨文演讲后给出民主党正式回应。她批评特朗普的经济政策"鲁莽"，指责他"偏袒亿万富翁以牺牲普通美国人为代价"。她说里根总统如果看到特朗普讨好普京"会在坟墓里翻来覆去"。此次回应引起广泛关注，使 Slotkin 迅速成为民主党的全国性面孔之一。

## 政治定位

Slotkin 被描述为**温和民主党人**（New Democrat），GovTrack 2022年评分称她是众议院中第五保守的民主党人。她认为安全和经济是民主党必须重新赢回的核心议题。

### 关键立场

| 议题 | 立场 |
|------|------|
| **国家安全** | **核心关注**。CIA+五角大楼背景，支持强势国防，认为"中产阶级衰退是美国最大安全威胁" |
| **经济** | 温和派。支持汽车工业（唯一投票阻止加州2035禁售燃油车的参议院民主党人）；支持工会 |
| **医保** | 支持ACA，反对全民医保（Medicare for All），支持 Medicare 按需购买选项 |
| **移民** | 相对温和。支持 Laken Riley 法案（少数投支持票的民主党参议员之一） |
| **枪支** | 支持背景审查和安全储存法（受2021牛津高中枪击案影响） |
| **身份政治** | 公开表示身份政治"应该像渡渡鸟一样灭绝"，主张关注独立美国人的普遍利益 |
| **外交** | 温和鹰派。支持对以色列军售但有条件；反对从叙利亚撤军；支持乌克兰 |

## 综合评价

Elissa Slotkin 是**参议院最新晋但也最具国家安全背景的成员之一**。她的职业生涯横跨CIA、国安会和五角大楼，三次前往伊拉克战区服役。

**独特定位**：她是新一代温和民主党人的代表——在安全议题上持鹰派立场，在经济议题上关注中产阶级，在社会议题上主张"超越身份政治"。她认为民主党必须重拾"阿尔法能量"，在国家安全和经济议题上展现出更强的领导力。

**对华方面**：她的立场以国家安全为出发点，支持与中国的激烈科技竞争但不一定支持完全脱钩。作为密歇根参议员，中国电动汽车对美国汽车工业的竞争也是她的核心关注。

**上升潜力**：2025年国情咨文回应显示了民主党将她视为未来人物的意愿。随着 Peters 于2027年退休，Slotkin 将自动成为密歇根资深参议员。
