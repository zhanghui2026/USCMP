## 政治履历

### 学术生涯

- **1977–1978**：罗格斯大学法学院讲师
- **1978–1983**：休斯顿大学法学中心，副教授（1981年获终身教职）
- **1983–1987**：德克萨斯大学法学院教授
- **1987–1995**：宾夕法尼亚大学法学院教授（William A. Schnader 商法讲席教授）
- **1995–2012**：**哈佛法学院教授**（Leo Gottlieb 法学讲席教授）；曾为哈佛薪酬最高的非行政教职

Warren 是破产法和商法领域的顶尖学者，发表12本书和100多篇文章。她与 Teresa Sullivan 和 Jay Westbrook 合著的《As We Forgive Our Debtors》成为破产法经典。她是美国法学院中少数在公立大学完成法学教育的哈佛终身教授。

### 政策参与

- **1995**：全国破产审查委员会委员
- **2005**：反对《破产滥用防止及消费者保护法》（后未能阻止通过）
- **2006–2010**：FDIC 经济包容顾问委员会成员
- **2008–2010**：**国会监督小组（TARP）主席**
- **2010–2011**：总统助理兼财政部特别顾问，负责建立**消费者金融保护局（CFPB）**

## 2020年总统竞选

Warren 在2020年民主党总统初选中一度成为领跑者（2019年秋），以"我有个计划"为口号，提出全面政策方案，包括财富税、全民儿童保育、绿色新政等。她在艾奥瓦州党团会议中获得第三名，新罕布什尔州第四名，后在"超级星期二"表现不佳后于2020年3月5日退选。

## 核心政策立场

Warren 是参议院中最具影响力的进步派（progressive）人物，以消费者保护和银行监管著称。

| 议题 | 立场 |
|------|------|
| **金融监管** | **核心议题**。创建CFPB；推动恢复Glass-Steagall法案；称"过大的银行就是过大到不能审判" |
| **税收** | 支持财富税（对资产5000万美元以上者征税2%）；提高企业和富人税率 |
| **医疗** | 支持"全民医保"（Medicare for All）过渡方案 |
| **教育** | 支持免学费公立大学、学生债务减免 |
| **住房** | 关注私募股权推高租金问题 |
| **外交** | 反战进步派。反对伊拉克战争，支持外交途径解决冲突；对以色列持批评态度（支持对以军售限制） |
| **社会议题** | 进步派。支持堕胎权、LGBTQ权利、种族平等 |
| **劳工** | 强有力支持工会、最低工资上调至15美元 |

## 对华立场分析

### 总体定位：经济民族主义与对华强硬派

Warren 是民主党内对华最鹰派的进步派人物之一。她将对中国立场融入她的经济民族主义框架，批评中国"操纵经济规则"。

### 关键对华事务

| 议题 | 立场 | 具体行动 |
|------|------|---------|
| **贸易** | 强硬 | 主张对华征收惩罚性关税，批评中国"窃取知识产权" |
| **科技** | 限制 | 支持芯片出口管制、审查TikTok等中国平台 |
| **人权** | 批评 | 关注新疆和香港议题 |
| **台湾** | 支持 | 支持美台关系，但持审慎态度 |

## 标志性事件

1. **"Nevertheless, she persisted"**（2017年2月）——在 Jeff Sessions 司法部长确认听证中，Warren 阅读 Coretta Scott King 批评 Sessions 的信件，被共和党以"违反参议院第19条规则"禁言。McConnell 的名言"她被警告了，她被解释了，但她仍然坚持"成为女性参政运动的标志性口号。
2. **"You didn't build that"**（2011年）——Warren 阐述"没有人靠自己致富"的视频在网络病毒式传播，后被奥巴马在2012年竞选演讲中引用。
3. **银行听证**（2013年至今）——Warren 在银行委员会听证中多次追问银行高管"为什么不做任何事？"，其质问视频经常获得数百万观看。

## 综合评价

Elizabeth Warren 是**美国参议院中最有影响力的进步派斗士**，也是消费者保护的标志性人物。从哈佛法学院教授到创建CFPB再到参议院，她的职业生涯始终聚焦于"系统是否对中产阶级公平"这个问题。

**对华方面**：她是民主党内将经济民族主义与对华强硬立场结合得最紧密的人物之一。她认为中国是"操纵规则"的典型，支持对中国商品征收关税和科技限制。

**影响力**：作为参议院银行委员会首席成员和民主党党团副主席，Warren 在金融监管、消费者保护和进步派政策上拥有重大话语权。她虽未赢得2020年总统提名，但其理念——财富税、反垄断、银行监管——在后来的民主党政策辩论中延续了深远影响。
